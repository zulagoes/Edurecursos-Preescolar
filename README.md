# 📚 EduRecursos Preescolar

## Descripción

EduRecursos Preescolar es un repositorio digital diseñado para organizar y compartir recursos educativos dirigidos a docentes y estudiantes de educación preescolar.

## Objetivo

Facilitar el acceso a materiales educativos digitales mediante una plataforma organizada, intuitiva y de fácil consulta.

## Recursos disponibles

- 📄 Guías didácticas
- 🎥 Videos educativos
- 🖼️ Imágenes
- 📊 Presentaciones
- 🎮 Recursos interactivos
- 📝 Evaluaciones
- 📚 Material complementario

## Usuarios

- Administrador
- Docentes
- Estudiantes

## Funcionalidades

- Buscar recursos.
- Descargar archivos.
- Clasificar por categorías.
- Evaluar recursos.
- Compartir materiales.
